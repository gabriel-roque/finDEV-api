## 📝 Minimal Requirements

- NodeJs 12.x

## 🚀 Install

```sh
$ git clone https://github.com/gabriel-roque/finDEV/
```

## 📝 Configure WebApp

```sh
$ cp .env.example .env
$ npm start
```

> Check in **development** at http://localhost:3333/

> Check in **production** at https://findev-api.herokuapp.com/
